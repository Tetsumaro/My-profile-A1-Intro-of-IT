A description of the skills, qualification and experience required for the position

The position requires a wide range of skills, qualification, and experience in developing and managing a web application service, including the front and backend programming skills as well as knowledge in a web server and database system. The advertisement indicates the following technical requirements.

Intermediate HTML and CSS knowledges

These programming languages are required to develop a user-friendly frontend interface.

An understanding of Webpack and Webpack CLI

Webpack is a tool to bundle “modules” which are various components of an IT project coded with JavaScript as programming language (https://webpack.js.org/concepts/ ). Webpack CLI is a type of add on for Webpack (https://www.npmjs.com/package/webpack-cli ).

Basic JavaScript knowledge including frontend frameworks such as Vuejs and libraries such as JQuery.

Frameworks refer to a set of codes written by others and consist of most common components used in a software development. Libraries are also a set of codes but can contain more variety of codes than frameworks (https://sofienebk.medium.com/what-is-the-difference-between-a-framework-and-library-2b712a1a1c41 ). Frameworks and libraries are tools to achieve conformity in coding as well as efficiency in a software development. With these tools, programmers can develop a software without writing commonly used codes from the scratch.

Basic GIT best practices

The position requires a good understanding with GIT, a version control system. (https://blog.devmountain.com/git-vs-github-whats-the-difference/ ).

Basic RESTful API Integration knowledge

RESTful API is a type of API based on an architectural style called REST(https://www.mulesoft.com/resources/api/what-is-rest-api-design ). API stands for application programming interface and is a set of protocols or coding information made publicly available by the application provider (https://www.mulesoft.com/resources/api/what-is-an-api). It enables users of one application to connect with another application. An example of API is a use of Google or Facebook user authentication system adopted by Amazon.

Intermediate PHP 7.0+

PHP is one of computer programming languages to develop the server side of web application.

Linux (Server OS, running on Amazon Web Services)

Linux is an open-source operation system which is commonly used to operate a web server (https://en.wikipedia.org/wiki/Linux ). Unlike more publicly known operation systems such as Windows and iOS, the user interface of most Linux versions (called distributions) is Command Line Interface (CLI) as opposed to Graphical User Interface (GUI). It means that it requires Linux user to make command by typing without using a mouse. It is understood that the company’s Linux runs on Amazon Web Services which is a cloud service offered by Amazon.

MYSQL 5.6

MYSQL is an open-source database system to store information gathered from a web server (https://en.wikipedia.org/wiki/MySQL).
